AI LifeBalance – Personalized Life and Productivity Assistant

AI LifeBalance is an intelligent assistant that helps users organize daily tasks, balance study and work with personal life, and maintain healthy habits. It adapts to each user’s energy levels, priorities, and deadlines to generate optimized daily schedules, while providing gentle reminders for breaks, hydration, and self-care.

Built with Python, Streamlit, FastAPI, and SQLite, the app combines AI-driven scheduling with simple productivity analytics to help users stay consistent and focused.

Key Features

Personalized Task Management: Add, update, and track tasks with priorities, categories, deadlines, and energy requirements.

AI-Driven Scheduling: Generate optimized daily plans tailored to your energy levels and available time slots.

Health and Well-being Reminders: Notifications for breaks, hydration, and self-care.

Productivity Analytics: Insights into task completion, category distribution, and streak tracking.

Secure Login: User credentials are managed via environment variables for privacy.

Interactive UI: Built with Streamlit, offering an intuitive and visually appealing interface.

Tech Stack

Frontend: Streamlit
Backend: FastAPI
Database: SQLite
AI Integration: LangChain, OpenAI/Google Generative AI APIs
Visualization: Altair, Matplotlib, Pandas
